Every letter and drink name is positioned within a div that has the proper stylistic classes applied.

Due to my lack of six letters, I have put the letters from my first name, i.e., S to the first row, which is my last name, KOTHA.

The drink names are listed in the second row.

Using inline CSS instead of a separate CSS file, I created a responsive grid layout using the Bootstrap classes (container, row, col-sm-2, col-md).

Rrow containing the drink names (col-md) will break first when the browser window gets smaller. This is because the drink names will change their width according to the available space, but the letters (col-sm-2) have a constant width.

As the available space decreases, the drink names will eventually wrap to the next line.
